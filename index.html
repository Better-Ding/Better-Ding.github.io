Research Focus

研究方向

在崔予文教授团队中，主要聚焦于铝合金等金属材料体系的智能建模与性能预测，结合机器学习方法与材料物理机理， 开展成分 - 工艺 - 组织 - 性能关系的定量建模研究。重点关注小样本数据条件下模型构建与泛化问题，通过引入物理约束、 数据增强与优化策略，提高模型对材料硬度、弹性模量等关键性能的预测能力，并探索智能算法在材料成分优化与材料设计中的应用潜力。

Selected Projects

项目经历

2025.06 - 2025.12 机器学习 + 相图预测

多元贵金属体系相边界的机器学习研究

  • 以 Pd-Pt 与 Au-Ag-Ge 体系为例，建立机器学习赋能的相图相边界预测模型。
  • 设计基于多活度导数信号几何联合的自动标记策略。
  • 结合主动学习方法（LC、Margin、Entropy）对相边界进行迭代优化。
  • 应用技术：边缘分割方法、不确定性采样、机器学习。
2024.08 - 2025.12 强化学习 + 材料设计

基于强化学习的材料设计方法研究

  • 为新型材料设计提出最优可行性方案，解决传统材料设计搜索空间巨大的问题。
  • 采用“模型预测 -> 实验推荐 -> 实验验证”的完整设计环路。
  • 智能体设计涵盖策略网络与目标网络，并计算 Q 值进行决策。
  • 应用技术：DQN、GPR。
2024.01 - 2024.05 性能预测 + 小样本建模

Al-Mg-Si 合金（6xxx 系列）机械性能预测：硬度和模量

  • 通过物理指导下的机器学习映射铝合金成分与性能之间的关系。
  • 使用 RF、SVM、GPR、MLP 等模型预测铝合金硬度和模量。
  • 使用 Optuna 进行超参数调优，并通过 SMOGN 缓解小样本问题。
  • 应用技术：Sklearn、Optuna、SMOGN。
2023.08 - 2023.10 深度学习 + 3D 表征

基于 3D CNN 的铝合金延性预测

  • 通过材料中的孔隙缺陷特征预测材料延展性。
  • 对样本进行 3 维重构建模，并将气孔映射输入模型。
  • 搭建 3D 卷积神经网络，并通过 Optuna 进行参数调优。
  • 应用技术：3DCNN、Optuna、Pytorch。
